# LRS Overlay

Этот репозиторий содержит исходный код Twitch‑расширения для игры «Земли былых легенд: Молчание богов».  Оверлей показывает зрителям сведения о героях и словарь терминов, а административная панель позволяет редактировать эти данные без прямого изменения JSON‑файлов.

## Что нового

- **Новый дизайн**: красно‑золотая гамма, декоративные линии, плавные тени и гладкие градиенты.  Карточки героев стали визуально привлекательнее.
- **Главный экран**: два больших раздела — «Герои» и «Словарь».
- **Адаптивность**: интерфейс подстраивается под мобильные устройства.  Панели стягиваются в столбцы на узких экранах, а блоки с характеристиками прокручиваются горизонтально при нехватке места.
- **Смена темы**: кнопка 🌙/☀️ переключает тёмную и светлую темы.  Все цвета задаются через CSS‑переменные.
- **Словарь**: кнопки увеличения и уменьшения шрифта, поиск по терминам и описаниям, фильтрация по буквам.
- **Админка**: вкладки для героев и словаря, формы редактирования, статусные кнопки, загрузка и скачивание JSON‑файлов.  Админка также поддерживает тёмную и светлую тему.

## Структура

- `index.html` — основная HTML‑страница оверлея, используемая новым манифестом.
- `viewer.html` — fallback‑страница для старых клиентов (подключает `scripts.js`).
- `viewer.js` и `scripts.js` — основная логика оверлея (классы, маршрутизация, обработчики).
- `styles.css` — полный набор стилей.  Не содержит `@import`, все переменные и правила заданы здесь.
- `editor.html` / `editor.js` — административная панель, позволяющая редактировать JSON‑данные.
- `admin.css` — стили админки, согласованные с общей темой.
- `data/` — папка с `heroes.json`, `glossary.json` и `settings.json`.
- `manifest.json` — файл манифеста Twitch.

## Обновление данных

1. Откройте `editor.html` через GitHub Pages или локально.
2. Отредактируйте героев или словарь.
3. Используйте кнопки «Скачать heroes.json» и «Скачать glossary.json» для сохранения изменений.
4. Замените файлы в `docs/data/` на полученные и сделайте коммит.
